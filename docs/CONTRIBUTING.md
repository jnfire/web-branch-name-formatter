# Contributing to Branch Name Formatter

We love contributions! Since this is a simple utility tool, the workflow is straightforward.

## 🛠️ Development Setup

1. **Clone the repo**
2. **Install dependencies**: `npm install`
3. **Run dev server**: `npm run dev`
4. **Run tests**: `npm run test`
5. **Type-check**: `npm run type-check`

## 🎨 Styling Guidelines
- Use **Vanilla CSS** within Vue components.
- Follow the **neutral/sober** design palette (grays, whites, blacks).
- Ensure support for both light and dark modes.

## 📄 Commit Messages
We follow a simple descriptive format in English:
- `feat: ...`
- `fix: ...`
- `refactor: ...`
- `docs: ...`
